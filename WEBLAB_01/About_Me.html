Web Developement Notes

Basics
  • The language being used is html
  • < >'s are called "tags" and they go around everything
  • Tags have opening and closing tags
  • html and css are used to code for websites
  • These basic text files of code make up of every website

Paragraph Text
  • Paragraph text is like a print statment without any variables
  • To create paragraph text you just type within a < p > and < / p >
  • To create a new line or "return" you use < br / > at the end of your line of code

Page Navigation
  • Page navigation helps the page user go where he or she wants by clicking a link
  • The link sends you to a specified spot on the page
  • The link is set to send you to what the name of your code is
  • So, to use page naviagtion you need to name where you want to go using "a name" function

Links
  • Links send you to places
  • Links can send you to places in the web, a different page on your website, or even a place on the page your on
  • Links show up as blue underlined text when on the website
  • They can be labed as different things that can take you to whatever place you tell it to

Images
  • When imputting an image you use "img src = the link of the photo"
  • You can adjust the hieght of width of the photo by setting "height/width = #"
  • The image needs to be in an image type file like jpg
  • The image also NEEDS to be in the same folder as the html being worked on, or it iwill not show up on the website
